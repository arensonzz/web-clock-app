## Requirements

* npm

## Running Server Locally

1. Install a HTTP server app to be able to serve static files.
    ```sh
    npm install -g http-server
    ```
1. Start the server.
    ```sh
    http-server
    ```
1. Access the application from browser
    ```sh
    http://localhost:8080
    ```
